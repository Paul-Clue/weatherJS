## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- Browser
- Code Editor
- NPM installed

### Setup

- Clone or download the repository.

### Install

- Run `npm install`.

### Usage

If you change anything in javascript files... you need to run `npm run build`.
